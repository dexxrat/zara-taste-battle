# Zara Taste Battle

**Interactive product comparison platform with merge-sort ranking algorithm | Pet Project**


---

## About

A full-stack web application for ranking Zara clothing items through pairwise comparisons. Users choose between two products in a tournament-style battle. The winner is determined by a custom O(n log n) merge-sort algorithm, producing a complete ranked list per subcategory with direct links to Zara.

Parsed Zara product data. Built as a solo pet project to practice full-stack development and algorithm design.

---

## How It Works

1. Select a clothing category and subcategory from Zara
2. Vote in head-to-head comparisons between two products
3. The backend applies a merge-sort ranking algorithm
4. View the final ranked top list with product links

---

## Tech Stack

- Java 21, Spring Boot 3.5, PostgreSQL
- React 19, TypeScript, CSS
- Custom O(n log n) merge-sort ranking algorithm
- Zara product data parsing
